SPAR structural analysis system consists of a collection of processors\nfor performing finite element analysis. The data generated by each of these\nprocessors is stored in a data file organized as a "library." Each data set,\ncontaining a two-dimensional table or matrix, is identified by a four-word\nname listed in the table of contents.\nThis report documents the contents of many of the SPAR data sets stored\nin the data base complex. The creating SPAR processor, number of rows and\ncolumns, and definitions of each of the data items, are listed for each data\nset. An example SPAR problem using these data sets is also presented. The\nSPAR input for the problem and resulting table of contents are included, so\nthat the user may reproduce these data sets. This information can be used\n(1) to understand more clearly and to use more productively the existing\nprocessors in the SPAR system, (2) to develop new SPAR processors, or (3) to\ninterface SPAR with another software system.\n\n\xe2\x80\xa2\n\n36\n\nREFERENCES\n1.\n\n2.\n\n\xe2\x80\xa2\n\nWhetstone, W.
