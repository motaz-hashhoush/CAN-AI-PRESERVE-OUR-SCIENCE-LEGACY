Justus\n\nEarth-GRAM Overview\n\xe2\x80\xa2\n\nGRAM is a Fortran software package that can run on a variety of platforms\nincluding PC\xe2\x80\x99s.\n\n\xe2\x80\xa2\n\nGRAM provides values of atmospheric quantities such as temperature,\npressure, density, winds, constituents, etc.\n\n\xe2\x80\xa2\n\nGRAM99 covers all global locations, all months, and heights from the surface\nto ~ 1000 km).\n\n\xe2\x80\xa2\n\nDispersions (perturbations) of these parameters are also provided and are\nspatially and temporally correlated.\n\n\xe2\x80\xa2\n\nGRAM can be run in a stand-alone mode or called as a subroutine from a\ntrajectory program.\n\n\xe2\x80\xa2\n\nGRAM07 is diagnostic, not prognostic (i.e., it describes the atmosphere, but it\ndoes not forecast).\n\n\xe2\x80\xa2\n\nThe source code is distributed free-of-charge to eligible recipients.\n\nGRAM07 Characteristics\n\xe2\x80\xa2 GRAM output comes from a database of atmospheric\nmeasurements.\n\xe2\x80\xa2 New measurements can be introduced using the\nAuxiliary Profile feature.\n\xe2\x80\xa2 Monte Carlo runs reproduced the observed mean and\nstandard deviations.\n\xe2\x80\xa2 The dispersions are pseudo-Gaussian distributed\n(except for pressure which is dominated by large-scale\ndisturbances).\n\xe2\x80\xa2 The small-scale dispersions have a Dryden power\nspectrum.\n\xe2\x80\xa2 The computed wind shears are consistent with those\nobserved at KSC.\n\nEarth-GRAM07 Output\nFor any position and time, Earth-GRAM provides\nmean and (optional) perturbed values density\nRelative humidity\nO3, N2O, CO, CH4, CO2, N2, O2, O, Ar, He, H, N\n\nGRAM does not provide global distributions of\ncloud cover, precipitation, visibility, or lightning\n\nGRAM07
